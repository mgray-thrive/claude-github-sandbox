# claude-github-sandbox

Throwaway test repo to validate the Claude Code → GitHub → Pages loop before migrating real projects.

- **Dummy content only.** No Thrive/customer data — that's why this one is public.
- `index.html` deploys to GitHub Pages via `.github/workflows/pages.yml` on every push to `main`.
- Real projects use **private ThriveNextGen** repos + **auth-gated** hosting, not public Pages.

Created 2026-06-07 as the migration warm-up.
